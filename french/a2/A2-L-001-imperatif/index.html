Étape 4 : Écoute, lis la transcription et note les verbes
Écris les verbes d'action que tu repères. Write the action verbs you spot.
Ouvrez le robinet. Mouillez vos mains et vos poignets. Appliquez du savon doux dans vos paumes de mains. Frottez vos mains en commençant par les ongles, puis les doigts, les espaces interdigitaux, les paumes des mains, les dessus des mains, les pouces, puis finissez par les poignets. Rincez vos mains sous l'eau. L'idéal étant de le faire les doigts orientés vers le haut pour que les impuretés soient évacuées avec l'eau qui coule vers le bas. Essuyez-vous les mains avec du papier que vous utiliserez pour refermer le robinet. Jetez ensuite ce papier dans une poubelle.

Étape 5 : La règle de l'impératif
Lis et écoute les quatre règles. Read and listen to the four rules.
1 · Emploi

Donner un ordre, un conseil, une instruction.

Give an order, advice, an instruction.

2 · Sans pronom sujet

Les pronoms tu, nous, vous disparaissent : « Ouvre le robinet », au lieu de « Tu ouvres le robinet ».

The pronouns tu, nous, vous disappear: “Ouvre le robinet”, not “Tu ouvres le robinet”.

3 · Les formes & le « s »

Les 3 formes (verbe mouiller) : Première forme mouille (-) · Deuxième forme mouillons (ons) · Troisième forme mouillez (ez)

Verbes en -er : pas de « s » à la 1re forme — sauf en -ir (partir : pars (s), partons, partez) et -re (prendre : prends (s), prenons, prenez).

-er verbs: no “s” in the first form, except with verbs ending in -ir (partir/pars, partons, partez) and -re (prendre/prends, prenons, prenez).

4 · Le pronom après le verbe

Trait d'union : « lave-toi les mains », « ferme-le » (le robinet).

Hyphen: “lave-toi les mains”; “ferme-le” (the tap).
